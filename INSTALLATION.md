# Jira Checker Plus - Installation Guide

## First Time Installation

1. Download the latest release:
   - Go to: https://github.com/anish274/jira-checker-plus/releases/latest
   - Download `jira-checker-plus-0.6.0.zip`
   - Extract the ZIP file

2. Install in Chrome/Edge:
   - Open `chrome://extensions/` (or `edge://extensions/`)
   - Enable "Developer mode" (toggle in top-right)
   - Click "Load unpacked"
   - Select the extracted folder
   - Done! Extension is installed

## Auto-Updates

The extension will automatically check for updates every 5 hours and update itself.

To manually check for updates:
1. Go to `chrome://extensions/`
2. Click "Update" button at the top

## Updating to New Version

### Automatic (Recommended)
- Extension updates automatically within 5 hours of new release
- No action needed

### Manual
1. Download new version from releases page
2. Extract ZIP
3. Go to `chrome://extensions/`
4. Find "Jira Checker Plus"
5. Click "Remove"
6. Click "Load unpacked" and select new folder
